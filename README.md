# Airline Selenium Automation Framework

This project is a Selenium WebDriver automation framework built using Java and Maven to automate testing of an Airline Web Application.

## Automated Scenarios
- Flight search
- Booking flow
- Passenger details validation
- UI validations
- Regression testing

## Tech Stack
- Java
- Selenium WebDriver
- Maven
- TestNG
- Eclipse

## Project Structure
- Page Object Model design
- Reusable utility classes
- TestNG based test execution

## How to Run
1. Clone the repository
2. Import as Maven project in Eclipse
3. Run testng.xml
